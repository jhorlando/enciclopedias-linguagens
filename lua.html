Curso de linguagem Lua

Breve histórico

A linguagem Lua foi criada em 1993 no Brasil, no Tecgraf/PUC-Rio, por Roberto Ierusalimschy, Luiz Henrique de Figueiredo e Waldemar Celes, com o objetivo de ser uma linguagem de script leve, rápida e fácil de integrar em outras aplicações. Surgiu da necessidade de substituir outras linguagens utilizadas em projetos da Petrobras, como SOL e DEL. Lua rapidamente se destacou por sua portabilidade, simplicidade e eficiência, tornando-se popular em diversas áreas, incluindo jogos, sistemas embarcados e aplicações web. 

Vantagens

1. Leveza e desempenho: Lua é extremamente leve e rápida, ideal para sistemas embarcados, jogos e aplicações que exigem eficiência.
2. Fácil de aprender: Sua sintaxe simples e direta facilita o aprendizado, mesmo para quem está começando na programação.
3. Alta portabilidade: Um mesmo script em Lua pode ser executado em diferentes plataformas sem alterações, o que a torna muito versátil.
4. Integração com outras linguagens: Lua foi projetada para ser embutida em aplicações escritas em outras linguagens, como C e C++, o que permite estender funcionalidades com facilidade.

Desvantagens

1. Ecossistema limitado: Lua possui uma comunidade menor e menos bibliotecas disponíveis em comparação com linguagens como Python ou JavaScript, o que pode dificultar o desenvolvimento de projetos mais complexos.
2. Gerenciador de pacotes problemático: O LuaRocks, principal gerenciador de pacotes da linguagem, pode apresentar dificuldades de uso, especialmente em sistemas Windows.
3. Pouco suporte para aplicações de propósito geral: Lua é excelente como linguagem embutida, mas não é tão adequada para desenvolvimento de aplicações completas ou sistemas corporativos.
4. Fragmentação da comunidade: Existem diferentes versões, ferramentas e distribuições de Lua, o que pode gerar confusão e dificultar a padronização em projetos maiores.

Aplicações

1. Jogos: Lua é amplamente utilizada no desenvolvimento de jogos, com suporte em motores como Unity, Corona SDK e LOVE2D. Ela é empregada para criar scripts de lógica de jogo, inteligência artificial, descrição da interface do usuário e manipulação de objetos. Exemplos de jogos que usam Lua incluem World of Warcraft, Angry Birds e Roblox.
2. Sistemas Embarcados: Lua é usada em sistemas embarcados, como middleware Ginga para TV digital.
3. Aplicações Científicas e Acadêmicas: Lua é aplicada em simulação, modelagem, visualização de dados e processamento de imagens.
4. Automação: Sistemas de automação de tarefas, como o Windower para o jogo Final Fantasy XI, utilizam Lua.
5. Bancos de Dados: A linguagem Lua é utilizada no Redis para estender suas funcionalidades, permitindo a execução de scripts personalizados dentro do banco de dados.
6. Aplicações Web: Frameworks web como o Lapis também empregam Lua.
7. Redes: Lua é usada em sistemas de rede, como CISCO e Nmap.
